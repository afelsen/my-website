Learn More About the Paint App

I made this demo using Google's Quick, Draw! dataset.

I use a custom PyTorch convolutional neural network to classify drawing. The network is trained on hundreds of thousands of 64x64 hand-drawn doodle from Quick, Draw! dataset. After drawing a doodle, the image is sent Flask, decoded and classified through a previously-trained model.

The website is built using Flask, hosted using Amazon Web Services. The code for this website can be found at GitHub.
